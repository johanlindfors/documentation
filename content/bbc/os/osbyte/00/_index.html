--- type: "manual" title: "&00 (0) OS Version" linkTitle: "0 OS Version" weight: -1 categories: - bbc osbyte: - hex: 00 int: 0 title: Get OS version entry: x: "0 execute BRK with message of OS version, <>0 RTS with OS version in X" exit: a: Preserved x: OS version y: Preserved c: Undefined compatibility: electron: true bbc: true master: true ---

This OSBYTE call returns the OS type in X or as a message via BRK if X=0 on entry.

Returned values

X OS version
0 OS 1.00 or Electron OS 1.00
1 OS 1.20 or American OS
