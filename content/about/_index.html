Why Project Area51

In the beginning each project had it's own domain, however whenever a new project was started it ended up sitting in a subdomain of another unrelated project. That domain usually ended up being called area51 simply to indicate that it was a secret project of some kind. After a while this became untenable with multiple instances spread over multiple domains so the original area51.onl domain was setup & the project got it's new name.

Since then the domain's changed name, splitting into area51.dev for development, area51.media for video & area51.social for social media.
