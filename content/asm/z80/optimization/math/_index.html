Basic Arithmetic

A=-B

A simple one, we want to set A to be -B.

The logical way is to load A with B then negate it:

{{< book/sourceCode>}} {{< book/asm op="LD" param="A,B" code="78" comment="Set A to B" >}} {{< book/asm op="NEG" code="ED44" comment="Negate A to get A=-B" >}} {{< /book/sourceCode >}}

But a quicker and shorter way is:

{{< book/sourceCode>}} {{< book/asm op="XOR" param="A" code="AF" comment="A=0" >}} {{< book/asm op="SUB" param="B" code="90" comment="A=0-B = -B" >}} {{< /book/sourceCode >}}
