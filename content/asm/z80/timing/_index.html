The Z80 processor executes instructions as a series of basic operations:

  • Memory access
  • I/O device access
  • Interrupt acknowledge

Machine Cycles

Each of these operations is known as a Machine cycle (M-cycle). An M-cycle however can take between 3 and 6 clock cycles to execute.
