--- type: "manual" title: "PLOT" linkTitle: "PLOT" description: "Set cursor location" weight: 0xFFF0 categories: - c64 - kernalAPI api: - name: "PLOT" addr: "FFF0" title: "Set cursor location" entry: "x": "Row to set if carry clear" "y": "Column to set if carry clear" exit: "x": "Current row if carry set" "y": "Current column if carry set" ---

This routine either reads or sets the text cursor's location.

Reading cursor location

A call to this routine with the accumulator carry flag set loads the current position of the cursor on the screen (in X,Y coordinates) into the Y and X registers. Y is the column number of the cursor location (0-39), and X is the row number of the location of the cursor (0-24).

Example:

        SEC           ; Set Carry flag to read the text position
        JSR PLOT
        STX curRow    ; X is the row
        STY curCol    ; Y is the column

Setting cursor location

A call with the carry bit clear moves the cursor to X,Y as determined by the Y and X registers.

Example:

        LDX #10       ; Set cursor to row 10 column 5
        LDY #5
        CLC           ; Clear Carry flag to set the position
        JSR PLOT
