--- type: "manual" title: "Here strings" linkTitle: "Here strings" description: "Sending a string into a pipe" categories: - linux - bash ---

Here strings are similar to Here documents but allows you to do this on one line:

{{< highlight go "linenos=inline" >}} command <<<"some string" {{< / highlight >}}

For example, the following commands do the same thing, count the number of words in a string:

{{< highlight go "linenos=inline" >}} echo "This is a test" | wc -w wc -w <<<"This is a test" {{< / highlight >}}

You can also pipe the output of commands as a here string:

{{< highlight go "linenos=inline" >}} ps -fe | wc -l wc -l <<<$(ps -fe) {{< / highlight >}}
