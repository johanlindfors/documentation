--- type: "manual" title: "Cron scheduler service" linkTitle: "Cron" tags: - go-kernel service ---

The CronService integrates the gopkg.in/robfig/cron.v2 scheduler into the microkernel.

To add the scheduler simply add it as a dependency to any of your own Service's:

Adding dependency

{{< highlight go "linenos=inline" >}}package example import ( "github.com/peter-mount/go-kernel" "github.com/peter-mount/go-kernel/cron" ) type Example struct { cron *cron.CronService } func (p *Example) Name() string { return "Example" } func (p *Example) Init(k *kernel.Kernel) error { service, err := k.AddService(&cron.CronService{}) if err != nil { return err } p.cron = service.(*cron.CronService) return nil }{{< / highlight >}}
