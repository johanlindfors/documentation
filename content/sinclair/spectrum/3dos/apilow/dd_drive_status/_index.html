Synopsys

Issue a sense drive status command.

Entry Parameters

Register Content
C Unit/head
Bits 0…1 Unit 0…3
Bit 2 Head
Bits 3…7 = 0

Exit Parameters

A = ST3 status register 3 of uPD765A
Always:
F BC DE HL IX corrupt, all other registers preserved.
