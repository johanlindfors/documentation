Synopsys

Write a sector

Entry Parameters

Register Content
B Page for 0xC000…0xFFFF
C Unit 0 or 1
D Logical track, 0 base
E logical sector, 0 base
HL Address of buffer
IX Address of XDPB

Exit Parameters

If OK:
Carry true
A corrupt
Otherwise:
Carry false
A = Error code
Always:
BC DE HL IX corrupt, all other registers preserved.
