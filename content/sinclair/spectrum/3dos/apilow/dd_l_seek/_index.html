Synopsys

Seek to required track. Retry if failure.

Entry Parameters

Register Content
C Unit/head: Bits 0…1 unit, Bit 2 head
D Track
IX Address of destination XDPB

Exit Parameters

If OK:
Carry true
A corrupt
Otherwise:
Carry false
A = Error code
Always:
BC DE HL IX corrupt, all other registers preserved.
