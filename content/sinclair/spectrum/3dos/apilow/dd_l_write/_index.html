Synopsys

Low level uPD765A write command.

  • Write data.
  • Write deleted data.
  • Format a track.
  • Scan equal.
  • Scan low or Equal.
  • Writes commands.
  • Writes data.
  • Reads results.
  • Motor must be running.

Entry Parameters

Register Content
HL Address of parameter block

Exit Parameters

HL = Address of result buffer in page 7
Always:
AF BC DE IX corrupt, all other registers preserved.
