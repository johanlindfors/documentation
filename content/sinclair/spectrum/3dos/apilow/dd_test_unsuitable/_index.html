Synopsys

Check that disk is suitable to write to. A single track disk will not work in a double track drive, and vice versa.

Entry Parameters

Register Content
C Unit 0 or 1
IX Address of XDPB

Exit Parameters

If OK:
Carry true
A corrupt
Otherwise:
Carry false
A = Error code
Always:
BC DE HL IX corrupt, all other registers preserved.
