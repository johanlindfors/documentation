Synopsys

Is the floppy disk drive interface present?

This information is also held by BASIC in bit 4 of the FLAGS3 system variable.

Entry Parameters

None

Exit Parameters

If present
Carry true
Otherwise:
Carry false
Always:
A BC DE HL IX corrupt, all other registers preserved.
