Synopsys

Log in a new disk. Initialise the XDPB. This routine does not affect or consider the freeze flag.

Entry Parameters

Register Content
C Unit 0 or 1
IX Address of XDPB

Exit Parameters

If OK:
Carry true
A = Disk type
DE = Size of allocation vector
HL = Size of hash table
Otherwise:
Carry false
A = Error code
DE HL corrupt
Always:
BC IX corrupt, all other registers preserved.
