Synopsys

Returns the current position in the file

Entry Parameters

Register Content
B File number 0…15

Exit Parameters

If OK
Carry true
A corrupt
E HL = File pointer 0x000000…0xFFFFFF (16Mb)
E is most significant byte, L least
Otherwise:
Carry false
A = Error code
E HL = Corrupt
Always:
BC D IX corrupt, all other registers preserved.
