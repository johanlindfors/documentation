Synopsys

Point at the XDPB for this drive. (The XDPB is required by the floppy disk driver routines.)

Entry Parameters

Register Content
A Drive letter A…P

Exit Parameters

If OK:
Carry true
A corrupt
IX = Address of XDPB
Otherwise:
Carry false
A=Error code
IX corrupt
Always:
BC DE HL corrupt, all other registers preserved.
