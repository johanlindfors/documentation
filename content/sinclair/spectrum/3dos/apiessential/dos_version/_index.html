Synopsys

Get the DOS issue and version numbers.

Entry Parameters

None

Exit Parameters

If OK:
D = Issue
E = Version (within issue)
Otherwise:
Carry false
A=Error code
Always:
BC DE HL IX corrupt, all other registers preserved.
