Synopsys

Delete an existing file.

The file must not be open on any file number.

Entry Parameters

Register Content
HL Address of filename, wildcards are permitted.

Exit Parameters

If OK:
Carry true
A corrupt
Otherwise:
Carry false
A=Error code
Always:
BC DE HL IX corrupt, all other registers preserved.
