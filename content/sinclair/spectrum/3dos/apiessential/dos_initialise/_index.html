Synopsys

  • Initialise +3DOS.
  • Initialise disk drivers.
  • Initialise cache and the RAMdisk.
  • All files closed.
  • All drives logged out.
  • Default drive A: (if disk interface present), else M:
  • Default user 0.
  • Retry count 15.
  • Error messages disabled.

Entry Parameters

None

Exit Parameters

If OK:
Carry true
A corrupt
Otherwise:
Carry false
A=Error code
Always:
BC DE HL IX corrupt, all other registers preserved.
