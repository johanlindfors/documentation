Synopsys

Write a byte to a file, advancing the file pointer in the process.

Entry Parameters

Register Content
B File number 0…15
C Byte to write

Exit Parameters

If OK
Carry true
A corrupt
Otherwise:
Carry false
A = Error code
Always:
BC DE HL IX corrupt, all other registers preserved.
